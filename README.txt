This package includes code for the SARDONICS sampler, the Intracluster Moves sampler as well as code for adapting these samplers by using Bayesian Optimization (aka GP-bandits).


INSTALLATION:

This package was compiled on a Linux machine. But it should be easily modifiable to run on a Macintosh. For simple installation, just run setup.sh. In case of an compilation error, please go through setup.sh and modify the relevant make files as needed.
